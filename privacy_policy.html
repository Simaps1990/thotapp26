THOT – Politique de confidentialité

Dernière mise à jour : 16/03/2026

1. Résumé

THOT est un carnet de tir numérique. Les données de l’application sont stockées localement sur votre appareil. THOT ne nécessite pas de compte utilisateur obligatoire.

2. Données auxquelles l’app peut accéder

Selon les fonctionnalités que vous activez, l’application peut accéder :

  • Ville / lieu saisi par l'utilisateur : uniquement si vous renseignez ou sélectionnez une ville dans le champ de lieu d'une session afin de récupérer des informations météo associées.
  • Microphone : uniquement si vous activez un mode de minuteur utilisant la détection sonore.
  • Notifications : uniquement si vous activez les rappels de péremption de documents.
  • Caméra : uniquement si vous choisissez "Prendre une photo" dans le sélecteur de fichiers pour ajouter une image à un setup, consommable ou cible.
  • Stockage local : pour enregistrer vos sessions, inventaire, documents et préférences sur l’appareil.

3. Microphone : pourquoi, quand et comment il est utilisé

Pourquoi : le microphone est utilisé pour permettre la détection d’un départ sonore (par exemple un tir) afin de déclencher/arrêter automatiquement le minuteur quand vous activez ce mode.

Quand : le microphone est utilisé uniquement :

  • lorsque vous sélectionnez un mode de minuteur avec détection sonore ;
  • et pendant l’exécution du minuteur.

Important : aucun accès au microphone n’est effectué en arrière-plan sans action explicite.

4. Données audio : stockage, envoi, partage

  • Stockage : THOT ne stocke pas d’enregistrement audio.
  • Envoi : THOT n’envoie pas de données audio sur Internet.
  • Partage : THOT ne partage pas de données audio avec des tiers.

La détection sonore repose sur des mesures instantanées du niveau sonore sur l’appareil.

5. Services tiers : RevenueCat (abonnements)

Pourquoi : RevenueCat est utilisé pour gérer les abonnements Premium (achat, restauration, validation).

Quand : uniquement si vous accédez aux fonctionnalités Premium ou restaurez un achat.

Données : identifiants d’abonnement et de transaction nécessaires à la gestion. RevenueCat ne reçoit pas vos données personnelles ni vos contenus (sessions, documents, images).

Sécurité : communications sécurisées; RevenueCat est certifié ISO 27001.

6. Transferts de données hors de l’appareil

THOT peut transférer des données hors de l’appareil uniquement dans les cas suivants :

  • Météo : nom de ville vers API Open-Meteo.com pour récupérer données météo publiques.
  • Abonnements : identifiants de transaction vers RevenueCat pour gestion des abonnements.
  • Sauvegarde/Restauration : si vous utilisez explicitement la fonction de sauvegarde vers vos propres services (cloud, email, etc.).

THOT n’envoie aucun de vos contenus personnels (sessions, documents, images) vers nos serveurs.

7. Stockage des données

Les données de l'application sont stockées localement sur l'appareil dans le sandbox isolé propre à THOT, conformément aux mécanismes d'isolation des applications iOS et Android. Aucune donnée n'est transmise à un serveur distant. Les pièces jointes (photos, PDFs) sont stockées dans le même sandbox isolé.

L'accès aux données peut être protégé par un code PIN personnel (stocké sous forme de hash PBKDF2-HMAC-SHA256 avec 100 000 itérations, sel aléatoire de 16 octets, comparaison à temps constant) et, optionnellement, par l'authentification biométrique de l'appareil (Face ID, Touch ID, empreinte Android).

8. Caméra : pourquoi, quand et comment elle est utilisée

Pourquoi : la caméra est utilisée pour photographier vos setups, consommables et cibles.

Quand : uniquement si vous choisissez l'option "Prendre une photo" dans le sélecteur de fichiers lors de l'ajout ou la modification d'une image.

Comment : le sélecteur de fichiers natif iOS est ouvert par file_picker; si vous sélectionnez l'appareil photo, iOS demande la permission caméra.

Données : seule l'image capturée est stockée localement dans l'application. Aucune capture vidéo ni utilisation en arrière-plan.

9. Notifications : rappels de péremption des documents

Pourquoi : les notifications servent uniquement à vous rappeler une date de péremption de document.

Quand : uniquement si vous avez défini une date de péremption sur un document et activé un rappel (par exemple 7, 30 ou 90 jours avant), puis activé le switch de notifications dans les paramètres de l’application.

Données : le rappel est planifié localement sur l’appareil à partir des informations déjà enregistrées dans l’application. Aucun envoi de données documentaires à des tiers n’est nécessaire pour cette fonctionnalité.

10. Ville renseignée par l’utilisateur et météo

THOT n’accède pas à la position géographique de votre appareil.

Si vous renseignez ou sélectionnez une ville lors de la création d'une session, l'application peut utiliser cette information pour :

  • valider le lieu sélectionné ;
  • récupérer les conditions météo associées à cette ville (température, humidité, pression, vent).

Cette fonctionnalité repose sur la ville que vous fournissez volontairement dans l’application, et non sur la géolocalisation automatique de l’appareil.

11. Retrait du consentement / suppression des données

Vous pouvez retirer votre consentement à tout moment :

  • Microphone : désactivez le mode de minuteur avec détection sonore et/ou retirez l’autorisation Micro dans les réglages iOS/Android.
  • Notifications : désactivez les rappels de péremption dans les paramètres THOT et/ou retirez l’autorisation Notifications dans les réglages iOS/Android.
  • Caméra : retirez l’autorisation Caméra dans les réglages iOS (pas d’accès caméra dans THOT sans action explicite via file_picker).
  • Ville / météo : vous pouvez cesser d'utiliser cette fonctionnalité à tout moment en ne renseignant plus de ville dans vos sessions.

Vous pouvez supprimer vos données :

  • via la fonctionnalité de suppression des données locales disponible dans l’application ;
  • ou en désinstallant l’application.
