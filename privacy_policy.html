THOT – Politique de confidentialité

Dernière mise à jour : 16/03/2026

1. Résumé

THOT est un carnet de tir numérique. Les données de l’application sont principalement stockées localement sur votre appareil. THOT ne met pas en place de compte utilisateur obligatoire.

2. Données auxquelles l’app peut accéder

Selon les fonctionnalités que vous activez, l’application peut accéder :

  • Localisation : uniquement lorsque vous appuyez sur un bouton de localisation/météo lors de la création d’une séance.
  • Microphone : uniquement si vous activez un mode de minuteur utilisant la détection sonore.
  • Stockage local : pour enregistrer vos séances, inventaire, documents et préférences.

3. Microphone : pourquoi, quand et comment il est utilisé

Pourquoi : le microphone est utilisé pour permettre la détection d’un départ sonore (par exemple un coup de feu) afin de déclencher/arrêter automatiquement le minuteur quand vous activez ce mode.

Quand : le microphone est utilisé uniquement :

  • lorsque vous sélectionnez un mode de minuteur avec détection sonore ;
  • et pendant l’exécution du minuteur.

Important : aucun accès au microphone n’est effectué en arrière-plan sans action explicite.

4. Données audio : stockage, envoi, partage

  • Stockage : THOT ne stocke pas d’enregistrement audio.
  • Envoi : THOT n’envoie pas de données audio sur Internet.
  • Partage : THOT ne partage pas de données audio avec des tiers.

La détection sonore repose sur des mesures instantanées du niveau sonore sur l’appareil.

5. Localisation et météo

Lorsque vous appuyez sur un bouton de localisation/météo, l’application peut utiliser la position pour :

  • proposer automatiquement une ville (géocodage inverse) ;
  • récupérer les conditions météo locales (température, humidité, pression, vent).

Aucune localisation n’est récupérée en arrière-plan en dehors de cette action explicite.

6. Retrait du consentement / suppression des données

Vous pouvez retirer votre consentement à tout moment :

  • Microphone : désactivez le mode de minuteur avec détection sonore et/ou retirez l’autorisation Micro dans les réglages iOS/Android.
  • Localisation : retirez l’autorisation Localisation dans les réglages iOS/Android.

Vous pouvez supprimer vos données :

  • via la fonctionnalité de suppression des données locales disponible dans l’application ;
  • ou en désinstallant l’application.
